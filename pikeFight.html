Pike fishing App. For 1-3 competing anglers

To Start press New and then enter settings symbol

  1. The number of anglers
  2. The Team name of the anglers
  3. The name of fishing event or lake

To enter a catch

  1. Double click the pike representing the angler
  2. Adjust the length
  3. Document by taking a photo
  4. Close by swiping left or right

To Make a report

  1. Adjust the map to a sutible size and position
  2. Swipe down the pull down menu select Make Report
  3. Share the report
  4. Sample report with the winner TJ. With the 4 largest counted (under 60 not counted)

Map controls

There is also an option where you can export a sea chart for printout in the settings page "Save Map View"
